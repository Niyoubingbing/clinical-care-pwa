# 临床病人管理助手 - 功能特性

## 项目概述
面向临床医护人员的移动端 PWA 应用，核心目标是提升查房效率、减少临床遗漏。所有数据存储在浏览器 IndexedDB，支持离线使用。

## 核心功能

### 1. 病人管理
- 增删改查病人信息（床号、姓名、医疗组、手术日期）
- 按姓名/床号搜索，按医疗组筛选（全部/解组/勇组）
- 排序模式切换（查房顺序 / 床号自然顺序）
- 查血记录录入（HB/WBC/PLT/K/Na/Cr/Alb/CRP）
- 文本批量导入（格式：-床号 姓名）

### 2. 查房排序（硬约束）
- 所有病人列表严格遵循预定义的 ROUNDING_ORDER 查房顺序
- 不在排序列表中的床号排在末尾，按字母数字自然顺序
- 适用范围：首页病人列表、交班小结、待办汇总

### 3. 智能提醒
- 查血超期提醒：>7 天橙色警告，6 天黄色提醒，自动添加"开查血"待办
- 换药到期提醒：根据换药频率自动添加"换药"待办
- 危急值预警：Hb<70、WBC>12 等异常指标标红
- 换药待办完成后自动更新上次换药日期

### 4. 待办系统
- 病人专属待办：快捷标签（换药、开术前、明天出院、康复会诊、会诊、复查）
- 自定义待办内容
- 完成状态切换，记录完成时间
- 已完成事项可折叠查看
- 通用备忘录（独立于病人的待办）
- 待办汇总：聚合所有未完成待办，按查房顺序排列

### 5. 每日工作小结
- 自动汇总今日动态（完成待办、录入查血、确认换药）
- 生成格式化文本（如：📅 日期 / 👤 床号 姓名 / ✅ 待办 / 💉 查血）
- 一键复制到剪贴板
- 病人顺序遵循查房顺序
- 次日自动清空

### 6. 数据管理
- JSON 完整导出/导入备份
- 文本批量导入同步（姓名存在→更新床号，不存在→新增，缺失→确认删除）
- 所有数据本地 IndexedDB 持久化

### 7. PWA 支持
- 可安装到主屏幕
- 离线访问
- 版本自动更新提示

## 设计风格
- 现代医疗简洁风，白/灰为主，青色为强调色
- 解组蓝色标识，勇组绿色标识
- 移动端优先，大触控区域（44x44px 最小）
- 适度动画（页面切换、待办完成、操作反馈）
